ChronicleTree Relationship Schema

A complete and structured overview of all supported family relationships and their validation rules.

Core Database Relationships

  • Parent / Child

    Direct biological link, stored bidirectionally.

    Stored

  • Sibling

    Shares parent(s); automatically generated.

    Stored

  • Spouse

    Marriage bond with status tracking.

    Stored

Marriage Status Types

  • Current / Ex-Spouse

    Active or ended marriage (divorce/separation).

    Status

  • Deceased / Late Spouse

    Spouse has died (widow/widower status).

    Status

  • Late Spouse

    Perspective label when spouse is deceased.

    Calculated

Sibling Categories

  • Full Siblings

    Share both biological parents.

    Calculated

  • Half Siblings

    Share one biological parent.

    Calculated

  • Step Siblings

    No shared biological parents.

    Calculated

Extended Family

  • Grandparents / Grandchildren

    Generation +2 / -2 from self.

    Calculated

  • Uncles / Aunts

    Parent's siblings.

    Calculated

  • Cousins

    Uncle/Aunt's children.

    Calculated

In-Law Relations

  • Parents-in-Law

    Spouse's parents (excludes ex-spouses).

  • Children-in-Law

    Child's spouses.

  • Siblings-in-Law

    Spouse's siblings.

Step-Family Relations

  • Step-Parents

    Parent's spouse (not biological).

  • Step-Children

    Spouse's children (not biological).

  • Step-Grandparents

    Step-parent's parents.

Stored Core relationship stored in database.
Calculated Dynamically inferred from core data.
Status A variation of a stored relationship.

Business Rules & Validation

Blood Relative Marriage Prevention

Prevents marriage between parents, children, and siblings.

Single Current Spouse Rule

Only one 'current' spouse allowed per person.

Automatic Sibling Generation

Sibling links are created when parent-child links are added.

Age & Temporal Validation

Validates marriage age and logical generation gaps.

Ex-Spouse Exclusion

In-law/step relations exclude ex-spouse connections.

Bidirectional Relationships

Core relationships create automatic reciprocal links.
